Må vi kunne noe teknisk?

Nei, siden er laget for at det skal føles enkelt. Dere kan også skrive at dere hjelper med oppstart hvis dere faktisk gjør det.

Passer dette for skolearrangement?

Ja, teksten på siden er laget så det passer elevkvelder, bursdager og lokale events.

Kan vi endre prisene senere?

Ja. Alt er bare vanlig HTML, så dere kan bytte tekst og priser direkte i fila.

Funker siden i GitHub Pages?

Ja. Dette er en statisk side i én fil, så den kan lastes opp direkte i repoet deres.
