Hvorfor dette funker bra på stand

Folk stopper lettere når det skjer noe. Karaoke gjør standen mer levende og gir dere en aktivitet som trekker oppmerksomhet.

🎤

Lav terskel

Folk skjønner med en gang hva de skal gjøre, og det er lett å bli med.

📺

Synlig aktivitet

Musikk og skjerm gjør at standen føles mer aktiv og trekker flere bort.

⚡

Enkel å kjøre

Siden fungerer som en enkel kontrollside uten backend og kan hostes rett i GitHub Pages.

Vanlige spørsmål

Dette er greit å vite før dere bruker siden på arrangementet.

Funker dette i GitHub Pages?

Ja. Alt ligger i én HTML-fil med vanlig CSS og JavaScript. Det gjør den enkel å laste opp i repoet deres.

Kan vi bytte sangene?

Ja. Sanglista ligger i JavaScript og kan byttes når som helst.

Blir køen lagret?

I denne versjonen lever køen bare mens siden er åpen. Dere kan senere utvide med lagring i nettleseren hvis dere vil.

Trenger vi backend?

Nei. Denne løsningen er laget for å være så enkel som mulig og passe rett inn i GitHub.
